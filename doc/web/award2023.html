    審査員講評 (2023年10月28日)

    ネタバレを含みますのでご注意ください。

    (受賞)(作品名)(講評)
    総合大賞 玻璃の鏡像は嘘を吐く ちょっぴり切り口の異なるミステリーノベルゲー。犯人を見つけることが第一目標でない点が上手く活かされていたように思います。 中々にぶっ飛びSFな世界観ですが、「決められた独自のルール」の中から真実を推察させていく流れが真摯だったので、充分王道ミステリーと呼べる作品だと思ってますね。 TRUE ENDを見つけようと血眼にさせられた時点でもうかなりの良作だなと思い、総合大賞に選出させていただきました。 そして何よりも…キャラの魅せ方が上手い！細かい突っ込みどころは全て流せるほど登場人物たちが皆魅力的で刺さりました。TRUE ENDの後日談とか見てみたいですね～。（WCE）
    作り込まれた世界観・背景設定とそこから展開されていく先の読めないストーリー、臨場感のあるBGM、ギャグとシリアスのバランスが取れたキャラクター達の掛け合い…どれをとっても高水準にまとめられているその完成度に思わず脱帽しました。クローズド・サークルは使えば何でも面白くなるわけという訳では勿論なく、それを活かせる腕が無ければ面白くならないのでここまでの作品を作れるのは本当に凄いことだと思います。クリックする手が止まらなくなるほどつい没頭してしまう魅力的なシナリオで、特殊な設定を活かした展開に終始ドキドキさせられっぱなしでした。キャラクターの表現も素晴らしく、イラストそのものも勿論ですが表情の描き方や口調に滲み出る性格など細部まで違和感無く作られていました。スチルも豊富で構図も練られており、シナリオの印象を引き立てていたと思います。素晴らしい作品をありがとうございました。（KCS）
    シナリオ賞 迷子と恋と獣人と ストーリーの大枠としては、主人公であるアイシャが異世界に迷い込み、出会った相手に恋をするというよくあるものですが、その土台を十分に活かしきった良い作品でした。 アイシャが想い人への恋心や元の世界にいる家族へのわだかまりに悩む様子は"迷子"というタイトルとも上手くマッチしており、 その気持ちに対する決断が後の展開に大きく関わるところはノベルゲームとしての良さを出し切っているように感じました。 また迷い込んだ異世界での生活に慣れるまでの１か月をOPムービーにしているところは見事な工夫だと思います。（WCE)
    要素同士が喧嘩せずにきちんと共存できていた素晴らしい作品でした。首尾一貫して読み応えもあり、特にオープニングの描写や序盤での異言語が通じない部分、中盤での恋愛関係への発展のさせ方など細かな部分で納得感を感じさせる配慮がされている点も見事でした。主要な登場人物も皆それぞれキャラが立っていて余計な要素や違和感も特に無くすんなり頭に入ってきて、作り方が上手いなと感じました。個人的には不動産組のキャラが良く立っていてお気に入りです。（KCS）
    シナリオ賞 最果ての懺悔室 一言で表すなら美麗！。音、絵、文章と全体的に綺麗だなと思わせる作品でした。分岐の無い一本道型の作品ですが、不思議と没入感はありましたね。 静かで不思議な世界観とキャラクター。はじめは独立しているかに見えた各章の物語が、終盤に向けてうっすらと繋がっていく展開。 まるで現代小説を読んでいるかのような、美しいシナリオでした。（WCE）
    タイトル画面の時点で、その綺麗な世界観に引き込まれてしまいました。どの星で起こる出来事にも重厚なストーリーが有り、そのどれもが心を突き動かされるようなものでした。 主人公達が宇宙の星々に出向き紡いだ様々な縁、それらが複雑に絡み合い結末が変化していくので、最後までドキドキしながらシナリオを読み進めていました。 シナリオのみならず、ドット調のイラストも最高でした。画力の高さはもちろん、章が変わるごとにテーマとなる色に合わせて全体の色も変化するなどの芸の細かいギミックも魅力的でした。（KCS）
    イラスト賞 人食い菊ちゃん タイトル画面の雰囲気が醸し出すグロ＆ホラー系な雰囲気とは裏腹に、本編は意外とコミカル（？）な短編百合ノベルゲー。 途中でクイズが始まったりとゲーム性も高くて面白かったですね。流し見させない一種の工夫なんでしょうか？ 中でも特に魅力として挙げたいのは女子二人の表情！臨場感のある表現が素晴らしいと思い、イラスト賞に選出させていただきました！（WCE）
    どこか和風ホラー的な怖さを感じさせる雰囲気に、一見合わないようなテイストのかわいいキャラクターが意外にもマッチしていてグッと引き込まれました。世界観・雰囲気作りが本当に良く出来ていて、背景の赤色も単調のようでありながら縁の下の力持ち的に雰囲気作りをかなり手伝っており良い味を出していると思います。クイズという個性的なシステムを取り入れている点もこのゲームのユニークさに貢献していて良かったと思いました。（KCS）
    イラスト賞 イルマナ 全編通してイラストの使い方が丁寧で感心させられました。登場人物たちの表情が見えないイラストがホラーな作風に合っていたのはもちろんのこと、 場面に合わせて細かく切り替わる背景や重要な要素を印象付けるシーンイラストの挿入など、読み手が自然と物語に入り込める工夫が散りばめられており、恐ろしさを身近に感じさせる良い作品でした。（WCE)
    モノクロチックなテイストのドット絵によりこのゲーム特有の世界観が作り出されており、その他の細かい部分を含めイラスト面での手の込みように驚きました。環境設定画面というゲーム本編ではない外側の部分すらもゲームの一部にしている点など、今回はイラスト賞として挙げさせて頂きましたが一つのゲームとして見ても全体的に良くまとまっていたと思います。（KCS）
    音楽賞 美しい断首 学校という閉鎖的な舞台かつ学生が断首されていくという暗いストーリーの裏で、不気味さを醸し出しながらも主張し過ぎることはない音楽は絶妙なバランスでした。 登場人物たちの個性も豊かで、ただ恐怖を煽るだけのホラーには収まらない魅力に溢れており、独自の世界観を構成出来ていて素晴らしかったです。（WCE)
    個性的で読ませる文章、女学校感あふれる可愛い女の子たち、陰鬱とした空気感とそれを演出するおどろおどろしい音楽、それらが合わさり唯一無二の世界観が表現されていました。 女学校独特の気品あふれる雰囲気とグロテスクのミスマッチがクセになります。新しい天使様を見つけるために生徒の首を断つ学園長、それを止めるために学園長殺害を目論む主人公達、とても奇妙な正義の話でした。 各人が己の正義を実行しようと生きる様は、とても美しかったです。（KCS）
    短編賞 コールセンターの鈴木さんとVtuberの佐藤さん 物語というのは基本暗い話の方が面白くしやすいです。ゲームであればなおさら先への興味を煽ろうと工夫して、未知を探る暗い話になっていきがちです。 そんな中、本作は短いながら徹頭徹尾明るく、クスッと笑える作品として走り切っています。尚且つ全体的に手堅くまとまっていてスラスラ読めるのが印象的でした。(WCE)
    短編賞 クソを集めて、巨大な畑を作るのだ！ この設定を思いつく発想力が凄い。ギャグみたいなスタートから、いつの間にやら壮大な計画へと流れていく展開も実に芸術的だと思います。 タイトル通り「クソを集める」ゲーム部分も良いエッセンスになっていました。おまけに農業の勉強にもなるという、栄養たっぷりな作品です。(WCE)
    短編賞 先客 ストーリー性は薄い超短編ですが、サウンドやグラフィックが醸し出す雰囲気だけでプレイヤーを引き込む演出力が凄いです。 多くを語らず絶妙に想像を掻き立てられる日本昔話風の土台が、上手くマッチしていたように思いますね。（WCE）
    短編賞 OGRETalk 最初はどのような展開に転ぶのかまるで予測できませんでしたが、物語が進んでいくにつれて明かされていく驚きの真実。そうきたか！と2、3回くらい言わせてくれる作品でした。 エンド回収後にルート分岐直前に戻る選択肢がある良心的なシステムも好印象です。（WCE）
    短編賞 Imperfect Blue ～Re:Start～ 超王道な恋愛ノベルゲーム、その直球さが逆に刺さるものがあり、夢中でプレイしてしまいました。フルボイス、豊富なキャラの立ち絵、まさかのOP・EDテーマ付き、本当に至れり尽くせりの作品でした。 声優さん達の声はどれもキャラにマッチしていたし、どのキャラも個性的で良かったです。細部にまで気遣いが行き届いていて、相当力を入れて作られていました。空羽がお気に入りです。（KCS）
    短編賞 クールで薬なストーカーと、無職な俺でギャルゲを作ろう！ シナリオがとても良かったです。文章に勢いがあり、テンポよく読み進められました。最初こそ、主人公の破天荒さに大いに笑いましたが、途中から複雑な感情がうずまき、笑うのを躊躇してしまう場面もありました。 彼らは、その人生に嫌気が差して、死にたくなっても、結局はそれを受け入れて生きていくしかない。その辛さと力強さの板挟みで、情緒がどうにかなってしまいそうでした。（KCS）
    短編賞 少女がひとり、生贄となる。 とても短い作品でしたが、生贄としてみんなに恩を返したいと願う村娘の生き様がありありと感じられて、とても良かったです。イラストも綺麗で、だんだんと日が昇る演出は、美しく、切なかったです。（KCS）
    短編賞 デスゲームおかあさんといっしょ 衝撃の展開で、思わずゾクッとしてしまいました。ギミックを巧みに使った演出がとても印象的で、話の緩急がものすごかったです。本当に魅せ方が上手でした。イラストも可愛かったです。（KCS）
    短編賞 クリムゾンレッドの死神 キャラの過去や確執など、細かい所がしっかりと描かれていました。エキサイティングな展開が多く、終始ワクワクしながらプレイしていました。 また、どの場面のBGMも雰囲気にぴったりで、没入感もバッチリでした。（KCS）
    短編賞 蝉時雨の外れで 変わった設定の恋物語で、斬新で面白かったです。生まれたてゆえの無邪気さも良かったです。蝉ということで、物語の序盤からなんとなく予感していたことが本当に起きてしまったときは、心が苦しかったです。 切なくも心温まる、いいゲームでした。（KCS）
    スポンサー推薦佳作 終わりの星紀行 舞台は宇宙航海時代だが、文化どうしの接触や消滅は私たちが20世紀までに経験してきたことであり、大変興味深いテーマだった。（LUXION SOFT LLC.）
    スポンサー推薦佳作 お願い!!シシクラ神社 演出の使い方、使いどころが秀逸だった。キャラクタをこれでもかというほど動かしていた。（LUXION SOFT LLC.）
    スポンサー推薦佳作 人狼ゲェムのフォーヴィスム 〜檻にいるのは獣か人か〜 システムの作り込みがとてもよかった。（LUXION SOFT LLC.）
    スポンサー推薦佳作 クズ石のポルカ 群像劇の表現がとてもよかった。（LUXION SOFT LLC.）

    暴力表現に対する特例 (2023年8月6日追記)

    • 当コンテストの応募期間中に、ふりーむ様の暴力表現に関するR18審査基準が変更されるという事態が生じました
    • R18表現に対するレーティングはゲーム業界全体として推進するべきことと考えます
    • 一方で、当コンテストへの応募に関しましては、当コンテスト開始時のふりーむ様の審査基準を前提としていたため、一部のクリエイタ様にとってフェアでない状況となりました
    • つきましては、本件に対する救済として、下記に該当する場合に限り、ふりーむ様以外での公開であっても、当コンテストに応募可能といたします
      • ふりーむ様への新規掲載申請で、暴力表現を理由に通過できなかった場合
      • 暴力表現があるため、ふりーむ様への新規掲載申請を自主的に見送った場合
      • すでにふりーむ様で公開している作品であるが、暴力表現を伴うため、自主的に他サイトで公開しなおした場合
